# Packaging Odysey

Etat actuel du packaging desktop pour Odysey.

Fichiers utiles :
- `ANTIVIRUS_WINDOWS.txt` : strategie pour limiter les alertes antivirus et SmartScreen
- `COMMANDES_WINDOWS.txt` : builds Windows avec installation, sans installation et dossier brut
- `MACOS_LINUX.txt` : procedure locale et CI pour macOS et Linux
- `GITHUB_ACTIONS.txt` : workflow GitHub Actions pour generer les builds multi-plateformes
- `STATUT_BUILDS.txt` : resume des builds reellement generes et verifies
